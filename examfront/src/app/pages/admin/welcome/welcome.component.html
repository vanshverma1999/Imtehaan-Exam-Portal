Welcome to Admin Dashboard

Following are the set of instructions you can perform as an admin :

1) You can add categories here

2) You can add quiz of particular category

3) You can customize quiz on the basis of : No. of questions, max marks and which category it belongs to

4) You can add question bank in a quiz and questions will appear randomly for each user

5) You can update and delete a quiz

6) You can add update and delete questions in any quiz

7) You can set availability of quiz as : Active / Not Active

8) You can add quiz of particular category
